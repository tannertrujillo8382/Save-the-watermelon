Save the Watermelon 🍉

Project Description:
Save the Watermelon is a terminal-based word-guessing game written in Python. The player must guess letters to reveal a secret word before the “slice counter” runs out. Guess correctly to save the watermelon, but if you run out of slices, the watermelon is sliced and you lose!

How to Play
Run the game.
A masked word will be displayed, e.g., _ a _ e.
Guess one letter at a time.
Correct guesses reveal letters in the word.
Incorrect guesses reduce your remaining slices (lives). You start with 6 slices.
Win by revealing all letters before slices run out.
Lose if slices reach 0.
After a game ends, you can choose to play again.
Features & Rules
Random secret words chosen from a word list.
Displays masked word with guessed letters revealed.
Input validation: only single letters, no duplicates, letters only.
Replay option after win or loss.
Core gameplay follows classic word-guessing mechanics.
